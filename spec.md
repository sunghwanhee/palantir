# 팔란티어 (Palantir) — 프로젝트 스펙

## 1. 프로젝트 개요

### 목적
글로벌 주식시장의 전체 흐름을 한눈에 파악하는 일일 갱신형 대시보드 구축

### 핵심 가치
- 시장 전체 흐름 파악 (매매 타이밍 참고)
- 강한 섹터 / 약한 섹터 즉시 식별
- 거시경제 지표(환율·금리·원자재)와 시장의 연관성 파악
- 뉴스 + 데이터를 한 화면에서 소화

---

## 2. 대상 시장

| 시장 | 포함 자산 |
|------|----------|
| 한국 | KOSPI, KOSDAQ |
| 미국 | S&P 500, NASDAQ, DOW |
| 글로벌 | 유럽(DAX, FTSE), 아시아(닛케이, 항셍, 상하이) |
| 암호화폐 | BTC, ETH (가격 + 24h 등락률만) |

---

## 3. 핵심 지표 (Core Metrics)

### 3-1. 주요 지수
- 현재가 (or 전일 종가)
- 전일 대비 등락률 (%)
- 1주 / 1개월 차트 (기본 기간)

### 3-2. 섹터별 시세 (히트맵 형태)
- 미국: Technology, Energy, Financials, Healthcare, Consumer Discretionary, Industrials, Materials, Utilities, Real Estate, Communication Services, Consumer Staples
- 한국: 반도체, 2차전지, 바이오, 금융, 에너지, 화학, 자동차, 엔터
- 표시 방식: 등락률에 따라 색상 강도 변화 (진한 초록 = 강세, 진한 빨강 = 약세)

### 3-3. 공포·탐욕 지수 (VIX)
- VIX 현재값
- 전일 대비 변화
- 해석 레이블 (예: Extreme Fear / Fear / Neutral / Greed / Extreme Greed)

### 3-4. 거시경제 지표
| 지표 | 내용 |
|------|------|
| 환율 | USD/KRW, EUR/USD, USD/JPY |
| 금리 | 미국 10Y 국채 수익률, 기준금리 |
| 원자재 | WTI 원유, 금(Gold) |

### 3-5. 암호화폐
- BTC, ETH 현재가 + 24h 등락률
- 별도 섹션으로 분리 (주식과 동등 비중)

---

## 4. 뉴스피드

### 수집 대상
- 주식 시장 뉴스
- 경제 지표 / 중앙은행 정책 뉴스 (Fed, 한국은행 등)
- 글로벌 시장 영향 뉴스

### 언어
- 한국어 + 영어 기사 모두 포함
- 언어 필터 버튼 제공 (ALL / KO / EN)

### 표시 형태
- 우측 패널에 스크롤 가능한 피드
- 기사당: 제목 + 출처 + 발행 시각 (KST 기준)
- 클릭 시 원문 링크로 이동

## 6. 데이터 갱신

### 갱신 주기
- 기본: 일일 1회 자동 갱신
- 수동 새로고침 버튼 제공

### 갱신 기준 시각 (KST)
- 오전 7:00 — 아시아 시장 종가 반영
- 오전 8:00 — 미국 전일 종가 + 거시지표 갱신
- 뉴스 — 페이지 로드 시 최신 기사 fetch

### 타임존
- 모든 시각 표시: KST (UTC+9) 기준
- 예: "Updated at 07:00 KST"

## 7. 데이터 소스 (무료 API)

| 데이터 | API 후보 |
|--------|---------|
| 주식 지수 / 섹터 | Yahoo Finance (`yfinance` Python 라이브러리) |
| 거시경제 지표 | FRED API (미국 연준 제공, 무료) |
| 암호화폐 | CoinGecko API (무료, 무제한) |
| 뉴스 | NewsAPI.org (무료 tier: 100 req/day) |
| VIX | Yahoo Finance (`^VIX`) |

### 무료 API 제약 사항 (엣지 케이스)
- Yahoo Finance: 비공식 API, 간헐적 차단 가능 → 실패 시 전일 캐시 데이터 표시
- NewsAPI 무료: 24시간 지연 데이터 가능성 → 유료 전환 시 즉시 개선 가능
- FRED API: 일부 지표 1~2일 지연 발표 → 최신 발표일 함께 표시

## 8. 휴장일 처리

### 동작 방식
- 시장 휴장일 여부를 자동 감지
- 휴장일: 전일 종가 표시 + "CLOSED" 배지 노출
- 예: `KOSPI  2,450  CLOSED (as of Dec 29)`

### 공휴일 데이터
- 한국: 한국 공휴일 캘린더
- 미국: NYSE/NASDAQ 공휴일 캘린더
- 글로벌: 각국 주요 거래소 공휴일

## 9. 트레이드오프 결정 사항

| 항목 | 선택 | 이유 |

| 갱신 주기 | 일일 | 무료 API 제한 내 안정적 운영 가능 |

## 10. 기술 스택 (추천)
