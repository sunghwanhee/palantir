# 팔란티어 프로젝트

## 프로젝트 개요
웹 크롤링 기반 데이터 수집 자동화 도구

## 커뮤니케이션 규칙
- 종결어미: 명사형 사용 (예: "~함", "~임", "~됨")
- `~합니다` / `~입니다` 형태 금지
- 간결하고 명확한 표현 사용

## 주요 파일
- `CLAUDE.md` : 프로젝트 전체 규칙 및 컨텍스트 정의 파일
- `spec.md` : 기능 명세 및 요구사항 참고 파일
- `lessons.md` : 작업 중 발생한 문제, 해결 방법, 학습 기록 저장 파일

## 작업 규칙
- 새로운 기능 추가 전 `spec.md` 참고
- 문제 해결 후 `lessons.md`에 학습 내용 기록
- 코드 변경 시 `spec.md`의 요구사항과 일치 여부 확인
